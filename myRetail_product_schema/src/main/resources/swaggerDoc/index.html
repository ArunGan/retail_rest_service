Schema definitions

global_response: object

Wrapper for the common response object

request_identifier: string (20 to 50 chars)

Unique productIdentifier for the request. Will be used to debug the issues.

explanation_code: string

Response Code, 'RT0000' is success code. Refer to API documentation section for all possible response codes

explanation_message: string

Response code description.

response_timestamp: string (date-time)

Time stamp of response in MST, format is yyyy-MM-ddTHH:mm:ss.SSSZ
